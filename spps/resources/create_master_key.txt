Usage: create_master_key.py [-Force] [-Relocation <path>]")

       create_master_key.py -Help | -?

-Relocation <path>
    Use this parameters to persist the master key at another location instead
    of the default '.sbbs' folder of the current users home.

-Force
    Use this parameter in case when a master key file already exists. With this
    parameter you agree to overwrite it.
